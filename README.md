# Secure Code Assessment

## Project Description
This project is a simple Python login system developed using SQLite.

## Features
- User Login
- SQLite Database
- Login Authentication

## Vulnerabilities Found
- SQL Injection
- Plain Text Password Storage
- Hardcoded Credentials
- No Input Validation

## Recommended Fixes
- Use Parameterized Queries
- Hash Passwords
- Remove Hardcoded Credentials
- Validate User Input

## Tools Used
- Python
- SQLite
- Visual Studio Code
